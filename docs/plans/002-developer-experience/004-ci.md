# GitHub Actions CI

Add CI workflow that runs on every PR and push to `main`.

## Pipeline

1. `swift build` — verify compilation
2. `swift test` — run test suite
3. `swiftformat --lint` — format check (no auto-fix)
4. `swiftlint` — static analysis

## Matrix

- macOS 14 + macOS 15 runners (catch SDK compatibility issues across versions)

## Files

- `.github/workflows/ci.yml`
